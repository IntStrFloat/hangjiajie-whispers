
# Лендинг "Мой опыт путешествия по Чжанцзядзе"

## Описание
Премиальный минималистичный лендинг для продажи PDF-гайда о путешествии по национальному парку Чжанцзядзе. Сайт создаёт атмосферу спокойствия и масштаба китайских гор, не перегружая пользователя информацией.

---

## Блок 1: Hero (Первый экран)
Полноэкранный immersive-блок с атмосферным фоновым изображением туманных скал Чжанцзядзе. Мягкий тёмный градиент снизу для читаемости текста.

**Элементы:**
- Крупный заголовок "Мой опыт путешествия по национальному парку Чжанцзядзе"
- Подзаголовок в одну строку
- Контрастная CTA-кнопка "Получить мой PDF-гайд"
- Лёгкий параллакс-эффект при скролле
- На мобильных — sticky-кнопка внизу экрана

---

## Блок 2: О Чжанцзядзе
Лаконичная секция с одним атмосферным фото и коротким текстом, передающим ощущение места.

**Особенности:**
- Плавное появление при скролле
- Минимум текста — акцент на визуале
- Лёгкий параллакс на изображении

---

## Блок 3: Превью PDF
Красивый mockup PDF-гайда на устройстве (планшет/телефон) с примерами страниц.

**Содержание превью:**
- Готовые маршруты
- Карты и ключевые точки
- Где теряется время
- Что стоит и не стоит делать
- Всё на основе личного опыта

**Эффекты:**
- Плавающий mockup устройства
- Анимированное появление тезисов

---

## Блок 4: Покупка
Спокойный выделенный блок с ценой и финальным CTA.

**Элементы:**
- Заголовок "Получить мой опыт в формате PDF"
- Описание "Один файл, который заменяет часы подготовки"
- Цена: 499 ₽
- Кнопка "Купить и скачать PDF"
- Подпись: "Мгновенный доступ • Без подписки • Офлайн"

**Оплата:**
- Интеграция со Stripe для приёма платежей
- Автоматическая доставка PDF после успешной оплаты

---

## Дизайн и стиль

**Цветовая палитра:**
- Туманно-серые (#E8EAED, #B0B8C1)
- Глубокие зелёные (#2D4739, #1A2F23)
- Холодные каменные оттенки (#6B7280, #4A5568)

**Типографика:**
- Крупные, уверенные заголовки
- Много воздуха между блоками
- Контрастные CTA-кнопки

**Анимации:**
- Параллакс на фоновых изображениях
- Плавающие элементы (mockup устройства)
- Fade-in с небольшим сдвигом при скролле
- Плавные hover-эффекты на кнопках

---

## Техническая реализация

**Адаптивность:**
- Mobile-first подход
- Оптимизация для смартфонов, планшетов и десктопа
- Sticky CTA на мобильных устройствах

**Производительность:**
- Lazy-load для изображений
- Оптимизированные анимации (CSS-based)
- Быстрая загрузка для мобильных сетей

**Интеграции:**
- Stripe для обработки платежей
- Lovable Cloud для backend-логики и доставки PDF
